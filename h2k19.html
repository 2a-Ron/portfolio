Project Horror: Halloween 2019


Report:
This project was thought of and created in just over one week. I decided to do a simple walkthrough for halloween, my version of a virtual haunted house concept. This project is based around overlap triggered events to create a spooky environment. The goal is simple, walk through each level to find the entrance to the next level, while trying to make it back to your apartment safe.

Features:
- Built for PC/Console
- Run/Jump mechanics no combat
- Various overlap events
- Bonus in level one: turn off the generator
- My first time making animated credits and loading screens
